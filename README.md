## PotionMod
### Introduction

PotionMod is a Minecraft fabric mod that adds new potions to minecraft.

## Features

Realized potions:

* Blindness potion
* Darkness potion
* Mining Fatigue potion
* Luck potion
* Unluck potion
* Haste potion
* Hero of Village potion

Also added custom recipes for each potion.

<details>
<summary>Recipes</summary>
  
- Flint + Awkward Potion -> Blindness Potion
  
- Crying Obsidian + Awkard Potion -> Mining Fatigue potion

- Sculk Shrieker + Awkard Potion -> Darkness Potion

- Obsidian + Awkard Potion -> Haste Potion

- Blue Orchid + Thick Potion -> Luck Potion

- Wither Rose + Thick Potion -> Unluck Potion

- Totem of Undying + Thick Potion -> Hero of Village Potion

</details>

## Installation

1. **Install Fabric**:
    - Download and install the [Fabric Installer](https://fabricmc.net/use/).
    - Run the installer and select the correct Minecraft version.

2. **Install Fabric API**:
    - Download the latest version of Fabric API and place it in your Minecraft mods folder.
    - Fabric API on modrinth: [Fabric API](https://modrinth.com/mod/fabric-api)

3. **Download PotionMod**:
    - Download the latest release of PotionMod from the [Modrinth](https://modrinth.com/mod/potion-mod/versions).
    - Place the downloaded .jar file into your Minecraft mods folder.
